###细节问题

1.单行注释/*=====***.css start=======*/       一定要做好注释工作（相当重要！！！）
2.在某一块往下定位时最多定位3块
3.左中右三块，命名left middle right;   若为左右两块，则命名left right
4.统一用怪异盒子模型（星宇写在公共类里）
5.除了header以外其他都包裹在main里面，通过main来进行设置最大宽度以及统一的背景
6.每一块都有自己的定宽
7.注意什么元素就专门做什么工作，比如包裹的就专门用来包裹，等等等等
